### Added
- Tasks: a built-in Linear-style planner. Projects with short keys mint identifiers like `SHARP-123`; tasks carry state, priority, assignee, labels, due dates, sub-tasks, comments, and an activity trail, with list and drag-and-drop board views plus a detail peek.
- Task ↔ chat bridging: bare identifiers in messages become clickable chips, any message converts into a task from its hover toolbar, and ⌘K jumps straight to `KEY-123`.
- GitHub sync for tasks: branches and pull requests mentioning an identifier link themselves to the task, move it to In Progress, and close it when the PR merges. One "Copy branch" button keeps the loop tight.
- Task notifications: assignments and comments land in the inbox and push with deep links.
- Sharpy, the workspace AI assistant: ask questions and get streamed answers grounded in your messages, docs, and tasks — with citations that link back to the source. Retrieval respects channel and doc permissions.
- Kanban boards inside channels, now with card checklists and richer properties.
- Passkey sign-in (WebAuthn) alongside passwords.
- Voice calls: customizable video backgrounds, on-screen annotations over screen shares, and Audio Aura avatar effects.
- A week time-grid for seeing events in context and dragging across a time range to schedule a meeting.
- Automatic composer focus when opening or switching chats, so typing can start immediately.
- Settings About tab showing the app version and per-deploy build id.
- For self-hosters: optional Drizzle Gateway for private database management.

### Changed
- Meeting creators can edit schedules, details, and attendee lists after an event is created.
- The app updates itself right after a deploy — no more stale PWA tabs.
- Calmer channel switching and cleaner chat chrome on phones.

### Fixed
- Google Calendar connections now request the identity scopes needed to reliably find the connected account email.
- iOS standalone/PWA viewport recovery after backgrounding, plus human-readable DM meeting titles.
